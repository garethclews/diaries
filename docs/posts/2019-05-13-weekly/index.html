I’ve had an awful lot of documentation to read through over the previous posts so this time I’ll throw some recommendations for different ways to ingest the content. This includes podcasts and screencasts for all the cool kids.

Podcasts

  • Data Skeptic: a good generalist podcast
  • Linear Digressions: this one is more software engineering-y than Data Skeptic
  • Partially Derivative: alcohol and data science, what could go wrong? This one has finished but all the old casts are still good
  • Vidya’s data hack

Screencasts/presentations

  • Destroy All Software – this screencast in particular talks about how to make functional code work for the real world by embedding it in an imperative shell
  • Edward Kmett – regular twitch streams from the person responsible for Lenses (coming soon to non-haskell languages near you)
  • Are you glue? Not just for software engineers, this is a talk from a former google employee about careers that’s getting rave reviews from everybody that sees it

Articles

  • CAD designs for geometric ML
  • Automate the boring stuff 2.0!!!!!!
  • The ‘pareto optimal gender pay gap’ (aka When Algorithms Turn Sexist)
  • Machine learning and inference of cause
  • A trio of posts about functional programming and how they’re better than the other paradigms because you can predict the future

The life-changers

As a counterbalance for the stupid Uber algorithm I’ll try to bring some joy by showing those advancements which are having real positive impacts on people’s lives

  • Identifying impaired speech to give those whose voice is hard to control a means to be heard more easily
  • AI for predicting cancer and personalising care

Papers

  • Adversarial examples as features – this is really interesting, how details imperceptible to humans but work against algorithms could be used as features is pretty cool
  • Holistic semi-supervised learning
