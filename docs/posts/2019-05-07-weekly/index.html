Yann le Cunn

I’ve called this by his name because it should be familiar to everybody. He posts an awful lot of monologues about the state of machine learning and his opinion on things on facebook and has now declared that we should stop using ‘unsupervised’ as a term for machine learning tasks and instead use ‘self-supervised’. I was surprised at this because I’ve been using self-supervised as a descriptor for about a year already!
